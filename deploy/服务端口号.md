## 1. 数据库常用的端口号

### 1.1 Mysql

```bash
    3306  : 用于 MySQL Classic 协议（服务器端口选项）
    33060 : 用于 MySQL X 协议（服务器 mysqlx_port 选项）
    33062 : 用于使用 MySQL Classic 协议的管理连接（服务器 admin_port 选项）
```
